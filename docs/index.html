Enterprise Features. Zero Cost.

Everything AG Grid Enterprise charges $999/dev/yr for — shipped free under MIT.

🔍 Sort, Filter, Search

Multi-column sort, 14 filter operators, header filters, full-text search with Ctrl+F.

📊 Pivot Tables

Row groups, column groups, value aggregation, grand totals — interactive pivot configurator.

⚡ Virtual Scroll

100K+ rows without lag. Renders only visible rows with configurable overscan.

📈 Sparklines

Line, bar, and area mini-charts inside cells. SVG-based, trend detection.

↩ Undo / Redo

Full edit history with Ctrl+Z/Y. Cell edits, row add/delete, paste — all reversible.

📋 Range Selection

Excel-like click+drag range selection. Copy range with Ctrl+C, paste with Ctrl+V.

✏ Inline CRUD

Double-click to edit. Auto REST API integration. Add, delete, import from Excel/CSV/JSON.

🧮 Formula Engine

SUM, AVG, IF, ROUND — Excel-like formulas per row, defined declaratively as props.

🎨 Rich Templates

Status chips, avatars, progress bars, ratings, flags, links, thumbnails, currency.

🌐 Framework Agnostic

Web component works everywhere. Official wrappers for React, Vue 3, and Angular 17+.

⚙ Built-in Configurator

Toggle features, pivot, groups, themes — all from an interactive side panel.

♿ Accessible

ARIA roles, keyboard navigation, screen reader support, focus-visible outlines.

Quick Start

Get up and running in under 2 minutes.

<script type="module">
  import 'https://unpkg.com/@zentto/datagrid@latest/dist/zentto-grid.js';
</script>

<zentto-grid
  columns='[{"field":"name","header":"Name","sortable":true},
            {"field":"price","header":"Price","type":"number","currency":"USD"}]'
  rows='[{"name":"Widget","price":9.99},{"name":"Gadget","price":24.50}]'
  show-totals enable-toolbar enable-clipboard
></zentto-grid>
npm install @zentto/datagrid-react
